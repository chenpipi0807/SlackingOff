# 五子棋摸鱼终端 (Gomoku Moyu Terminal)

一个伪装成IDE界面的局域网五子棋游戏，让你可以在工作时间"合理"摸鱼。

## 功能特点

- 🎭 完美伪装成 VS Code IDE 界面
- ⚫ 命令行式五子棋游戏（简化命令 @字母）
- 🌐 支持局域网双人对战
- ⚡ 实时 WebSocket 通信

## 安装步骤

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 启动服务器：
```bash
python app.py
```

3. 访问游戏：
- 本机访问: http://localhost:5000
- 局域网访问: http://你的IP地址:5000

## 查看本机IP地址

Windows:
```bash
ipconfig
```
查找 "IPv4 地址" 那一行

## 游戏命令（简化版）

所有命令都是 `@字母` 格式，快速输入！

- `@j <名字>` - 加入游戏（例：@j 小明）
- `@s` - 开始游戏（需要2人）
- `@p <行> <列>` - 下棋（例：@p 7 7 表示中心位置）
- `@b` - 查看棋盘
- `@l` - 查看玩家列表
- `@m` - 查看历史记录
- `@c` - 清屏
- `@h` - 显示帮助

## 游戏流程

1. 两个玩家分别输入 `@j <名字>` 加入游戏
2. 任意玩家输入 `@s` 开始游戏
3. 轮流使用 `@p <行> <列>` 下棋
4. 先连成5子者获胜！

## 坐标说明

- 棋盘大小：15x15
- 坐标范围：0-14（用十六进制 0-E 表示）
- 中心位置：@p 7 7
- 示例：`@p 0 0` 左上角，`@p E E` 右下角

## 注意事项

- 真正的游戏在下方终端中进行

- 后端: Flask + Flask-SocketIO

- 通信: WebSocket
